# Screenshots wanted

Drop PNGs into this folder with the names below. The main README references them at these exact paths.

| filename | what it should show |
|---|---|
| `hero.png` | The dashboard's "money shot" — Advisor page mid-generation, with a recipe draft pane visible, or the Launch page with one healthy launch streaming logs. Wide aspect ratio (≥1600 px wide). |
| `boxes.png` | Boxes page with a clusters card section above the boxes table. Two or three boxes registered, at least one cluster of ≥2 nodes. |
| `box-detail.png` | A single box detail page showing the connection form (with cluster ChipInput pinned), capabilities (GPU model + VRAM), and status pill. |
| `launch.png` | Launch tab with the chip filter row visible, a target picked, recipe selected, and an active launch streaming logs below. |
| `advisor.png` | Advisor page: HF browser on the left, model detail in the middle, the cluster topology card visible, and Claude streaming a recipe draft. |
| `recipe-diff.png` | Recipe versions / diff view — two-column diff with the AI-assist "apply to form / save as new" controls. |
| `hf-browser.png` | Hugging Face browser with filters expanded (pipeline tag, library, sort) showing real model cards. |

PNGs ≥ 2× retina pixel density (so 2400 px wide for hero, 2000 px for the rest) read crisp on GitHub when scaled.
